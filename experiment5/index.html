Description

This experiment is an evolutionary generator that attempts to best replicate a set of Rorschach inkbots. Along the way, it generates entirely new inkblot-looking images!

Technical

We draw a couple hundred black and white ellipses on half of the canvas, mirroring them on the other half. Every frame, their size, position, and rotation is randomized. Each time, the resultant image is compared to the inspiration, generating a score. If this score is better than the best score, a new image is saved as the best one.
